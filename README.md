# Sistema de Gestão de Hotel

> 🎓 **Projeto Acadêmico**
> Repositório destinado aos projetos e atividades desenvolvidos para o curso de **Ciência da Computação da PUC**.

Sistema de gerenciamento hoteleiro desenvolvido na linguagem **C**.

Este projeto tem como objetivo simular as operações e o fluxo de controle de um hotel, como o registro de hóspedes, gerenciamento de reservas e controle de ocupação dos quartos. O desenvolvimento focou na aplicação prática de conceitos fundamentais da linguagem C, como o uso de structs, ponteiros e organização da lógica de software.

## 📂 Estrutura do Projeto

Os arquivos fonte principais do sistema estão localizados dentro do diretório `Hotel Projeto/`. A aplicação funciona através de um menu interativo no terminal, que permite ao usuário navegar facilmente pelas funcionalidades de cadastro e administração.

## 🚀 Como Compilar e Executar

Para rodar este projeto, você precisará de um compilador C (como o `gcc`) instalado no seu sistema.

Abra o terminal, navegue até a pasta correspondente e execute:

```bash
# 1. Entre no diretório do projeto
cd "Hotel Projeto"

# 2. Compile o código-fonte
gcc hotel.c -o sistema_hotel

# 3. Executar (Linux ou macOS)
./sistema_hotel

# 3. Executar (Windows)
sistema_hotel.exe
